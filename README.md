# Startup Funding Success Probability

## Problem Statement
The objective of this project is to identify factors associated with startup funding success using statistical analysis, probability analysis, and data visualization.

## Dataset Description
- Total Records: 500 Startups
- Features:
  - Industry
  - Funding Rounds
  - Funding Amount
  - Valuation
  - Revenue
  - Employees
  - Market Share
  - Profitable
  - Region
  - Exit Status

## Statistical Methods
- Mean
- Median
- Standard Deviation
- Correlation Analysis
- Probability Analysis
- T-Test

## Key Findings
- Overall Success Rate: 30.4%
- FinTech showed highest success probability.
- Asia showed highest regional success probability.
- Revenue significantly influences startup success.

## Visualizations
- Industry-wise Success Probability
- Region-wise Success Probability
- Funding Distribution
- Correlation Heatmap

## Recommendations
- Invest in high-revenue startups.
- Focus on FinTech and E-Commerce sectors.
- Consider profitability while evaluating startups.
